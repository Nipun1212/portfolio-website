This project was created because there was no available api for Anthropic's Claude 2 LLM. This lightweight python package reverse engineers the api that the claude 2 website uses and provides users with the ability to use the chatbot in the form of an API. People can use it for any number of use cases.
The user can simply extract the session cookies from their website and enter them to start chatting with the Claude 2 model.
It allows the users to -
1) Talk to the model
2) Start a new conversation
3) Delete conversations
(adding upload files feature soon)

The package gets updated constantly to add any new features that anthropic might release. Stay tuned :)

This is a project that was released as a pip package on PyPI. You can easily download it - pip install claude2
